Computational Linear Algebra Course

Institution: National Institute of Technology Calicut (NIT Calicut)

Instructor: Joe Cheri Ross

Course Description

This course introduces students to the fundamentals of computational linear algebra, focusing on the practical applications of linear algebra in various computational tasks. Topics covered include matrix operations, linear systems, eigenvalues and eigenvectors, numerical methods, and applications in data science, machine learning, and computer graphics.

Course Objectives

  • Understand the basic concepts and techniques of linear algebra.
  • Apply linear algebra methods to solve computational problems.
  • Implement algorithms for matrix operations and numerical methods.
  • Analyze and interpret results obtained from computational linear algebra techniques.

Prerequisites

Students are expected to have a strong background in calculus, algebra, and programming.

Course Structure

This course consists of lectures, tutorials. Assessment will be based on assignmentsand examinations.

Syllabus

Vectors & Geometry: Vectors & vector operations - Vector length, angle, and dot product - Matrix and matrix operations - Linear transformations

Linear Systems and Subspaces: Systems of linear equations, Gaussian elimination, LU Decomposition - Matrix and vector norms - Condition number - Least square approximation - Elementary matrices and matrix inverses - Subspaces, spans and linear independence - Bases and rank

Matrices: Co-ordinate systems - Determinants - Eigen values and Eigen vectors - Power method - Diagonalization

Advanced topics: Vector spaces and subspaces - Matrix decomposition - Orthonormal bases - Gram-Schmidt
