3. Prove: If \( V,W \in \Reals^n\) are unit vectors, then \(\vert V.W \vert \leq 1 \)
Answer: use Cauchy-Schwarz inequality \(\vert V.W\vert \leq \|V\| \|W\|\)

4. Let \(V, W \in \Reals^n \) be vectors that are orthogonal to each other. Prove that \( \|V+W\|^2 =\|V\|^2 + \|W\|^2 \)
Answer: \(\|V+W\|^2= (V+W).(V+W)\)
\(= (V.V) + 2(V.W) + (W.W)\)
\(= \|V\|^2 + \|W\|^2 \) Since V and W are orthogonal to each other \(V.W =0\)

5. For the given vector \( V= (v1, v2, v3)\), transform it to \(V'= (v2, v3, v1) \). Is this a linear transformation? If yes, find the transformation matrix T.
Answer: Yes. \( \begin{matrix} 0 & 0 & 1 \cr 1 & 0 & 0 \cr 0 & 1 & 0 \end{matrix} \)
